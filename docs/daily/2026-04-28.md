## 완료된 작업

### 1. 하드코딩된 Agent 제거 → 동적 JSON 기반 전환

**문제**: `adk_agents/` 폴더에 하드코딩된 `chatbot_xxx_adk` Agent들이 존재했고, JSON으로 추가한 챗봇은 이름에 하이픈(-)이 있어 ADK 규칙 위반으로 에러 발생.

**해결**:
- 모든 `chatbots/*.json` 파일의 ID를 하이픈 → 언더스코어로 변경
  - 예: `chatbot-hr` → `chatbot_hr`, `chatbot-hr-policy` → `chatbot_hr_policy`
- 하드코딩된 Agent 폴더들을 `adk_agents/backup/`으로 이동
  - `chatbot_company_adk/`, `chatbot_hr_adk/`, `chatbot_tech_adk/`, `architecture_agent/`, `implementation_agent/`
