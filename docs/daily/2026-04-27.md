#### Bug #1: `string indices must be integers`
**에러 메시지:**
```
TypeError: string indices must be integers
```

**원인:**
- `create_agent_tool()`에서 `chatbot_id`(str)를 `create_agent()`에 전달
- `create_agent()`는 `chatbot_def`(dict)를 기대

**해결:**
```python
# 수정 전 (에러)
agent = self.create_agent(chatbot_id)

# 수정 후 (정상)
chatbot_def = self._get_chatbot_def(chatbot_id)
agent = self.create_agent(chatbot_def)
```

#### Bug #2: `invalid agent name`
**에러 메시지:**
```
Found invalid agent name: `chatbot-company`.
Agent name must be a valid identifier.
```

**원인:**
- ADK Agent 이름에 하이픈(`-`) 사용 불가
- 언더스코어(`_`)만 허용

**해결:**
```python
# Agent 이름 변환
agent_name = chatbot_id.replace("-", "_")
```

class SubAgentFactory:
    def create_agent_tool(self, chatbot_id: str) -> AgentTool:
        chatbot_def = self._get_chatbot_def(chatbot_id)
        agent = self.create_agent(chatbot_def)
        return AgentTool(agent=agent)
    
    def create_root_agent_with_tools(self, chatbot_id: str) -> Agent:
        # 하위 Agent를 Tool로 등록

        return Agent(name=chatbot_id, tools=tools, ...)

#### 2. DelegationRouter
```python
class DelegationRouterAgent:
    async def route_and_stream_with_tools(...):
        root_agent = self.factory.create_root_agent_with_tools(chatbot_id)
        runner = Runner(agent=root_agent, ...)
        
        async for event in runner.run_async(...):
            if event.content and event.content.parts:
                for part in event.content.parts:
                    if part.text:
                        yield part.text
```
